About Me

Halo! Nama saya Rahma Khairani Panggabean.Saya adalah mahasiswi dari kampus Politeknik Ganesha Medan, Saya disini sedang belajar pengembangan web. Website ini saya buat sebagai proyek pribadi untuk menampilkan informasi tentang diri saya sendiri menggunakan HTML, CSS, dan JavaScript.

Dengan menggunakan Bootstrap, saya belajar banyak tentang cara kerja komponen UI seperti navbar, card, grid system, dan masih banyak lagi. Saya berharap website ini bisa menjadi langkah awal saya untuk mendalami dibidang pengembangan web.
